# dda-pallet-commons
Common utils for dda-pallet

## compatability
dda-pallet is compatible to the following versions
 * pallet 0.8
 * clojure 1.7

## Features
* Encrypted Credential Managament
* Inspecting pallet plans for tests
* Normailizing stevedore scripts for tests
